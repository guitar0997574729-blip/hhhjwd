# 🚀 HHH คอมมู - Quick Start Guide

## ⚡ 5-Minute Setup

### Step 1: Test Locally
1. Open `discord-widget.html` in your web browser
2. Click the ⚙️ button at bottom-right
3. Enter password: `369`
4. Update your server info and click "บันทึกและอัปเดต"

### Step 2: Get Your Discord Guild ID
1. Open Discord
2. Enable Developer Mode: Settings → Advanced → Developer Mode
3. Right-click your server name → Copy Server ID
4. Paste into "Guild ID" field in the admin panel

### Step 3: Update Invite Link
1. Go to your Discord server settings
2. Create an invite link (never expires)
3. Copy the full link
4. Paste into "Discord Invite Link" in the admin panel

### Step 4: Deploy to Web (Choose One)

#### **EASIEST: Vercel** ⭐
```bash
npm install -g vercel
cd "C:\Users\Acer\Downloads\ดิส"
vercel
```
✅ Done! You'll get a live URL

#### **GitHub Pages**
1. Create a GitHub account (if needed)
2. Create a new repository
3. Upload `discord-widget.html` as `index.html`
4. Go to Settings → Pages → Enable GitHub Pages
5. Your site is live!

#### **Netlify Drop**
1. Go to [drop.netlify.com](https://drop.netlify.com)
2. Drag your `discord-widget.html` file
3. Get instant live URL

## ❓ Common Issues

### Widget shows 0 members?
1. Check your Guild ID is correct
2. Ensure widget is enabled in Discord (Server Settings → Widget)
3. Clear browser cache

### Formatting looks weird?
1. Use Chrome or Firefox
2. Clear cache (Ctrl+Shift+Del)
3. Check mobile responsive

### Can't edit settings?
1. Make sure password is "369"
2. Check console for errors (F12)
3. Try different browser
